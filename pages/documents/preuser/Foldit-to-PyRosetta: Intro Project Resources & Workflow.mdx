# Foldit-to-PyRosetta: Intro Project Resources & Workflow

> Pre-user, Docking, Foldit, Modeling, PyRosetta

Follow along and explore the steps of this interactive workflow and resource hub by using Foldit, PyMol, and PyRosetta to evaluate a protein system that you are interested in! 

This workflow will give you the opportunity to investigate mutations in a protein system that you think might optimize, disrupt, or affect the system in some capacity!
