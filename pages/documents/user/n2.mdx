# Boot Camp: COVID-19 Evolution and Structural Biology 

> Pre-user, User, Foldit, Modeling, Nucleic Acids

June 2020 Boot Camp for undergraduate researchers held virtually to understand how the protein evolved during the first six months of the COVID-19 pandemic by exploring amino acid sequence and 3D atomic-level structure using various structural bioinformatics tools.
